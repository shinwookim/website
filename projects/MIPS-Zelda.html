An Assembly Adventure Game

MIPS RISC-V ASM

This is a 2D adventure game made in the style of the old Legend of Zelda games. The game was written entirely in MIPS assembly from scratch and is designed to be run on the MARS assembler. The game implements everything from a variety of weapons choices, functioning enemy AI, and a score system.
