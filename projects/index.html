Analysis of Sorting Algorithms

Using a formal approach, analyzed the run-time and space requirements for five sorting algorithms: QuickSort, Median of Three QuickSort, RandomPivot, Median of Five QuickSort, and MergeSort. Then, implemented these algorithms in Java and verified the results empirically.

Algorithms Java

CPU2224

A logical circuit design for a 16-bit CPU based on the Harvard computer architecture that shares many characteristics with early MIPS processors. CPU2224 supports ALU operations, load-stores, input-output, and control flow operations including conditional jumps, as well as function call instructions (like jump-and-link). The circuitry was created using Logism, a free educational tool for designing and simulating logic circuits; it supports loading and executing programs written in an MIPS-like assembly.

Digital Logic Logism

An Assembly Adventure Game

A 2D adventure game made in the style of the classic Legend of Zelda game. This game was written entirely in MIPS assembly from scratch using the MARS MIPS assembler. Within the game, there is a variety of weapon choices, functioning enemy AI, and a score system.
